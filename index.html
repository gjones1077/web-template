Video Effects with Cuda

In this project I implemented convolutions with image kernels on a video file. The image kernels were used for blurring and edge detection effects. Basically each pixel has an image kernel applied to it through each frame in a video using threads on the gpu allocated in user defined blocks. The kernel accounts for when there are more pixels than threads available on the GPU with striding.

NBA Central Division Database

With a group of 4 others, we created a database for every team in the NBA central division. For each of these teams we created five tables: players, stats, physical attributes, history and contracts. These tables provided crucial information about each team and the players that are on that team. The database can be used to analyze which players had the best statistics from the past season as well as what players were the best return on their salary.
